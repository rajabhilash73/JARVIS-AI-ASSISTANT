# JARVIS-AI-ASSISTANT
Jarvis Personal AI Assistant

Jarvis AI – Your All-in-One Personal Assistant 🤖✨
Jarvis AI is a powerful and customizable desktop assistant built using Python. Inspired by the iconic AI from Iron Man, this project integrates voice recognition, text-to-speech, a smart GUI, and ChatGPT capabilities to help you manage daily tasks, access apps, retrieve information, and much more—all through natural conversation.

🧠 Key Features
🎙️ Voice Input & Output – Communicate with Jarvis via speech, and get intelligent voice responses.

💬 ChatGPT Integration – Leverage the power of OpenAI to have human-like conversations.

🖼️ Custom GUI – A clean, animated graphical interface with assistant status and real-time interactions.

🔍 Search Anything – Get instant answers, summaries, or search results using AI.

🌦️ Weather Updates – Ask for real-time weather reports of any city.

🎵 Music Playback – Play your favorite songs or playlists with voice commands.

🗂️ App Launcher – Open commonly used applications like Notepad, Chrome, Calculator, etc.

📅 Date & Time – Ask for the current date, time, or set reminders.

🛠️ Extensible & Modular – Easy to add more commands and functionalities.

🛠️ Tech Stack
Language: Python

Libraries: speech_recognition, pyttsx3, openai, requests, datetime, and more

AI: OpenAI’s GPT models (ChatGPT)

GUI: Tkinter with animation and assistant visuals

🚀 Getting Started
Prerequisites
Python 3.10.10

OpenAI API Key (for ChatGPT integration)
